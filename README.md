# Gnosia-Game
그노시아 마피아 게임을 로컬 및 외부연결로 같이 놀수있는 게임

# 준비물

1. nodejs

# 설명서

1. npm i 명령어를 입력

2. npm start 명령를 줘서 서버를 엽니다

3. 사이트 접속 시작

즐겁게 플레이하세요 친구와 같이


# 주의사항

출처를 밝히지 않고 사용시 저작권으로 신고넣을껍니다 

저작권신고해당해서 채널 정지당하는건 제 책임이 아닙니다

역으로 저작권 이용해서 고소하지마세요 저희는 AI로 만든 작업물을 전부 보유하고있습니다
